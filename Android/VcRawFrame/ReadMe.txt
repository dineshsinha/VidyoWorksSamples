VcRawFrame android sample
Using static view to implement custom renderer
Usage of VIDYO_CLIENT_IN_EVENT_MUTE_DEFAULT_RENDERING to mute default renderer
Usage of VIDYO_CLIENT_IN_EVENT_STOP_WATCH_VIDEO_SOURCE / VIDYO_CLIENT_IN_EVENT_START_WATCH_VIDEO_SOURCE to control static watch
Usage of VIDYO_CLIENT_OUT_EVENT_VIDEO_FRAME_RECEIVED to view raw frame

Build instructions in - Android Build Instructions.docx in this folder.
